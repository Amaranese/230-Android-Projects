# Tuition-Notes-of-Student
This project is based on the tracking of students who go to tuitions. This app will have three modules that are the student, the tutor, and the parents. This app will contain all the details of the student, like when they had gone for classes and when they didnâ€™t. Also, the tutors will give them remarks on a daily basis
